    什么是Hexo

    Hexo是一个简洁而高效的博客框架；使用markdown渲染引擎将Hexo主题生成静态页面

    Hexo资料

    Hexo基本使用

    • Hexo安装npm install hexo-cli -g
    • 创建Hexo项目hexo init [folder]
    • Hexo项目启动hexo server
    • Hexo项目部署hexo deploy

    Hexo集成Github Pages

    目标是将hexo生成静态文件并发布到github仓库中进行解析

    • 创建github仓库

    • 创建Hexo本地项目

    • 安装Hexo部署插件npm install hexo-deployer-git --save

    • 修改_config.yml部署相关信息

    • 发布Hexo静态页面hexo deploy
