    IOC原理

    IOC即控制反转，又被称为依赖注入，其目的是解耦Bean之间的依赖关系；在org.springframework.beans和org.springframework.context包是Spring框架的IoC容器的基础; BeanFactory接口提供了一种能够管理任何类型对象的高级配置机制，而ApplicationContext则是BeanFactory的超集，目的是添加一系列企业级开发套件
